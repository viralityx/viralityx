Calculation engine — formulas encoded exactly from Excel
Z-Score
(athlete_value − db_mean) / db_stdev
Only computed when stdev ≠ 0. Filtered by gender + age group.
Database mean
AVERAGE(metric | gender + age_group match)
Matches Excel FILTER logic — gender × 9 age buckets.
Quartiles
Q1 = QUARTILE(filtered, 1)
Q3 = QUARTILE(filtered, 3)
25th and 75th percentile of the filtered cohort.
% of database avg
athlete_value / db_mean
Displayed as percentage alongside Z-score on dashboard.
Torque
force_lbs × moment_arm_ft
Auto-computed for all shoulder, hip, knee, ankle, elbow tests.
FBW / TBW
FBW = force / body_weight
TBW = torque / body_weight
Per PATIENT Strength_Map result type column.
DSI
dynamic_force / isometric_force
Pulled from Strength_Map column H per test.
TK / Mound ratio
tall_kneeling_mph / mound_mph
THROWERS only — derived from throwing velocity inputs.
% change from initial
(current − initial) / initial
Tracker views — compares current test to first recorded session.
Bilateral asymmetry
(stronger − weaker) / stronger × 100

Auto-generated summary text (e.g. "Tall Kneeling: Z = −2.08 → Needs Improvement"). Clinician notes section. PDF-export button.
S-08 Report approval dashboard Replaces manual email workflow
Manager view of all pending reports. Shows computed Z-scores, classification badges, and "Needs Improvement" flags. Actions: Preview report, Approve & Send, Hold. Eliminates manual email chain.
S-09 Excel data import Migrates all existing workbooks on day one
Upload .xlsx files — auto-detects THROWERS / PLYO / PATIENT by sheet names and column headers. Parses Bridge Intake Responses → client profiles. Parses Input Data Here / INPUT DATA HERE → all historical sessions linked to client + date. Skips rows where "Remove From Central Database" = "remove" or Pain = "yes" (imported with flag). Preview modal: client count, session count, date range. Duplicate detection: same client + same date = update, not duplicate. PATIENT: imports ROM sheet, PRO sheet, body-region sheets 11–20.
