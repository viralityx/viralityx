Ad Creative Guidelines

What works on Meta for local service marketing agencies:

🎥 Video Testimonials

30-60 second clips of real clients talking about results. "Before ViralityX we had no leads, now we're booked out." These outperform static images 3-5x on Meta.

📊 Before/After Results

Side-by-side screenshots showing a client's analytics growth. Numbers are attention-grabbing in the feed. "From 12 leads/month to 87 leads/month in 60 days."

🎓 Educational Carousel

"5 Reasons Your Social Media Isn't Getting Leads" — educational content that positions ViralityX as the expert. Last slide = CTA to book a call.

🔥 Behind-The-Scenes

Show the team working, strategy sessions, content shoots. Humanizes the brand and builds trust before the first conversation.

Content → Ad Strategy (The TikTok Playbook)

Don't make ads. Make TikToks. Then boost the ones that work.

🎬 "Day in the Life" Content

"Day in the life running a marketing agency" — behind the scenes of client work, content shoots, strategy sessions. These build trust and curiosity at scale.

🔥 Spark Ads (Boost Winners)

Post 5 organic videos per week. Whichever gets the most organic traction in 24-48 hours → boost it as a Spark Ad with a lead form attached. Let the algorithm pick your creative.

The 5-2-1 Rule

Post 5 organic TikToks per week. 2 will get decent views. 1 will pop off. Boost that one. Repeat every week. This is cheaper and more effective than creating ads from scratch because the algorithm has already validated the content.

When to Scale Each Channel

✅ Scale Signal

CPA is under $500 and ROAS is 3x+ for 30 consecutive days. Increase budget by 20% per week (not all at once — Meta/Google algorithms need gradual increases).

⚠️ Hold Signal

CPA is $500-800 and ROAS is 2-3x. Optimize before spending more: test new creative, adjust audiences, improve landing page conversion rate, or tighten follow-up sequences.

🛑 Pause Signal

CPA is over $800 or ROAS is under 2x for 14+ days. Pause the underperforming campaigns, diagnose the issue (creative fatigue? wrong audience? landing page problem?), fix it, then relaunch.

🚀 Launch Checklist

Step-by-step — what to do before you spend your first dollar on ads

Pre-Launch: Foundation (Week 1-2)

  • Run ghl-final-setup.html — create missing contacts, calendar, Deal Size field
  • Build all 6 automation workflows in CRM (specs in your build report)
  • Complete A2P 10DLC registration for SMS compliance
  • Connect Google Business Profile for reputation management
  • Build Meta landing page in CRM page builder (one CTA, form, social proof)
  • Build Google landing page in CRM (match search intent, dedicated URL)
  • Set up UTM tracking framework across all channels
  • Connect Meta Lead Ads via native CRM integration
  • Set up TikTok webhook or Zapier connection to CRM
  • Test every automation end-to-end (submit test lead on each channel, verify SMS fires)

Meta Launch (Week 2-3)

  • Install Meta Pixel on CRM landing page + website
  • Create Custom Audiences: website visitors (180d), social engagers (90d)
  • Upload Closed Won contacts from CRM → create 1% Lookalike
  • Create Campaign: Leads objective → 2 ad sets (Lookalike + Retargeting)
  • Create 3-4 ad creatives: video testimonial, before/after results, educational carousel
  • Set daily budget: minimum $50/day ($1,500/mo)
  • Let ads run 3-5 days before making ANY changes (learning phase)
  • After learning phase: kill ads with CPA over 2x target, scale winners

Google Launch (Week 3-4)

  • Set up Google Ads conversion tracking on CRM form submission
  • Create Search Campaign with 3 ad groups (brand-adjacent, problem-aware, service-specific)
  • Add negative keyword list (free, cheap, DIY, jobs, etc.)
  • Write 3 responsive search ads per ad group (15 headlines, 4 descriptions each)
  • Set up call extensions, sitelink extensions, callout extensions
  • Set daily budget: minimum $33/day ($1,000/mo)
  • Enable Google Ads → CRM conversion reporting via GCLID
  • Review search term report weekly — add negatives, find new keywords

TikTok Launch (Week 2 onward — organic first)

  • Start posting 5 organic TikToks per week (day in the life, tips, results)
  • Track which videos get 1,000+ views in first 24 hours
  • Set up TikTok Ads Manager + install TikTok Pixel on landing page
  • Create Spark Ad campaign using top-performing organic video
  • Attach lead gen form or link to CRM landing page
  • Set daily budget: $20/day ($500/mo) minimum
  • Rotate new Spark Ads weekly based on organic performance
  • Scale winners — if a Spark Ad gets CPL under $8, increase budget 20%

Next Channels to Add (After 90 Days)

Once Meta, Google, and TikTok are producing consistent ROI, layer in these channels to widen the moat:

📬 Postcards / Direct Mail

Hyper-local targeting. Send branded postcards to businesses in your service area. Include QR code → CRM landing page with UTM: ?utm_source=postcard. Track exactly how many scan and convert.

📋 Flyers / Door Hangers

Low-cost, high-touch. Target specific business districts or commercial areas. Same QR → landing page → CRM flow. Great for building local presence.

🏗️ Billboards

Brand awareness play. Use a memorable URL or phone number that routes to CRM. Track via vanity URL: viralityx.com/grow → landing page with utm_source=billboard.

🤝 Sponsors / Partnerships

Sponsor local events, podcasts, or business groups. Each sponsor deal gets a unique UTM link or promo code tracked in CRM. Attribute revenue back to the partnership.
